**优化 babel-loader**

- 开启缓存，`loader: ['babel-loader?cacheDirectory']`
- 排除范围，include 和 exclude 两者选一个即可

```js
module.exports = {
  entry: {
    index: path.join(srcPath, 'index.js'),
    other: path.join(srcPath, 'other.js')
  },
  module: {
    rules: [
      {
        test: /\.(js|jsx)$/,
        loader: ['babel-loader?cacheDirectory'], // 开启缓存
        // 排除范围，include 和 exclude 两者选一个即可
        include: srcPath,
        exclude: /node_modules/
      },
    ]
  }
}
```
