# 装饰器模式

**介绍**

- 为对象添加新功能
- 不改其原有的结构和功能



**演示**

![](https://github.com/negrochn/study-imooc/blob/master/255/img/uml-decorator.jpg)

```js
class Circle {
  draw() {
    console.log('画一个圆')
  }
}

class Decorator {
  constructor(circle) {
    this.circle = circle
  }
  draw() {
    this.circle.draw()
    this.setRedBorder(this.circle)
  }
  setRedBorder(circle) {
    console.log('设置红色边框')
  }
}

const c = new Circle()
const d = new Decorator(c)

d.draw()
```
