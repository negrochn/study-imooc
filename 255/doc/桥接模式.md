# 桥接模式

**介绍**

- 用于把抽象化与实现化解耦
- 使得两者可以独立变化



**演示**

```js
class Color {
  constructor(name) {
    this.name = name
  }
}

class Shape {
  constructor(name, color) {
    this.name = name
    this.color = color
  }
  draw() {
    console.log(`画一个颜色为${this.color.name}的${this.name}`)
  }
}

// 测试
let red = new Color('red')
let yellow = new Color('yellow')
let circle = new Shape('circle', red)
circle.draw()
let triangle = new Shape('triangle', yellow)
triangle.draw()
```



**设计原则验证**

- 抽象和现实分离，解耦
- 符合开放封闭原则
